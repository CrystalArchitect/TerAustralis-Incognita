# Archive

Superseded material kept for provenance (Repository principle 3). Nothing
here is maintained, nothing here is production, and **nothing new should be
built on anything in this folder** — it exists so history stays checkable,
not so it stays usable.

Internal paths and claims inside archived material describe the repository
*as it was* and are deliberately left unfixed.

## Contents

| Folder | What it is |
|---|---|
| `legacy/crystalcore-app/` | The pre-monorepo application (Lumina's ancestor with its own site, batch launchers, and docs) — superseded by `src/apps/lumina/` and `src/site/` |
| `2026/local-snapshot-2026-07-17/` | A dated working-tree snapshot from the machine migration, including the crystalcore v0.13 recovery state |

New archive material lands under a dated folder (`2026/…`) or `legacy/`,
with a line added to this table saying what it is and what superseded it.
