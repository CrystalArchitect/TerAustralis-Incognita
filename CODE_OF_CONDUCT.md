# Code of Conduct

This project is small, early-stage, and independently built and self-funded
(see [README.md](README.md)) — stewarded by one maintainer, the
CrystalArchitect. This document is short and tailored to that scale rather
than adapted from a larger community's template; it says what actually
applies here.

## Scope

This applies anywhere people interact around this project — issues, pull
requests, discussions, and any community space linked from README.md.

## Expected behavior

- Assume good faith. Disagree about code, ideas, and content — not about
  people.
- Give and take feedback plainly and respectfully. Blunt is fine; contemptuous
  isn't.
- Respect [CONTRIBUTING.md](CONTRIBUTING.md)'s Belt-Three law: keep Science,
  Story, and Vision content straight, and don't overclaim what's built.

## Cultural respect

Some of this project's Vision-layer content (`mythos/`) draws on Dreamtime
Songlines as narrative and architectural metaphor. Contributions that touch
this material should treat it as collaboration and honour, not extraction —
no claimed ownership of restricted cultural knowledge, no invented "sacred"
detail, no treating real Indigenous culture as a skin for the fiction. This
is the same rule CONTRIBUTING.md states for the Story label; it applies to
how contributors talk about the material, not just what they write in it.

## Unacceptable behavior

Harassment, personal attacks, discriminatory language or jokes, doxxing, and
sexualized content or advances have no place here. Neither does deliberately
misrepresenting what this project's code does or claiming Built status for
work that isn't.

## Enforcement

The maintainer decides on and enforces consequences, which may range from a
warning to removal from project spaces, at their discretion given the scale
of the project. There's no committee — one person is reading reports and
making the call.

## Reporting

Report privately via GitHub → **Security** tab → **Report a vulnerability**
(the same private-reporting channel used for security issues — see
[SECURITY.md](SECURITY.md)), or, if that's unavailable, open an issue titled
"conduct contact request" **without details** and a private channel will be
arranged. Expect acknowledgement within 7 days.

*Non Solus.*
