# CrystalCore.Lattice

**Locked name** ([Constitution §1](../governance/Constitution.md)): the
substrate — multi-AI weave, memory, ontology, activation.

## Honest status: designed, not built

The Lattice is the project's original design for a coherent multi-node
mind: a Weave Map registering every AI node, time-stamped **Lattice deltas**
recording each substantial piece of agent work, a contradictions ledger, an
exported `corpus/` for local models, and an activation sequence measured by
gates G0–G5.

**None of that machinery exists.** The entire Lattice today is one design
sketch: [`lattice/BOOT_STATUS.md`](lattice/BOOT_STATUS.md), which says so
itself. The gate board it refers to (`singularity-protocol.md`) was never
written. Nothing in this repository measures, records, or enforces any
Lattice state. The Constitution's implementation note (2026-07-21) marks
§3, §4, and §8 as Vision accordingly.

## What exists instead

The working practice that replaced the designed machinery:

- Substantial mythos content goes to `mythos/content/`.
- Dated status changes land in
  [`Roadmap.md`](../governance/Roadmap.md#recently-landed).
- Agent work is an ordinary commit with an honest message, through review.
- Contradictions go to the maintainer directly.

And the *spirit* of the Lattice — many AIs, one canon, closed feedback
loops — has a small, real embodiment in the Starline Weaver
([`AI-Weave.md`](AI-Weave.md)): not a planetary substrate, just a bus where
agents must label their truth or go unheard.

## The definition worth keeping

The Constitution's §4 definition of "singularity" stands whether or not the
Lattice is ever built: a coherent multi-node mind sharing one canon, one
constitution, and closed feedback loops — explicitly **not** uncontrolled
self-replication, weaponization, or bypassing human arbitration. If the
Lattice is built someday, it gets built under that definition, gate by
measured gate, with the ADR trail to show for it.
