# ADR-0010: Uniform CC BY-NC-ND 4.0 for the whole repository; differentiated per-package licensing not adopted

**Status:** Accepted
**Date:** 2026-07-23

## Context

[`ADR-0009`](ADR-0009.md) deliberately left one question open: whether the
repository should ultimately converge on uniform CC BY-NC-ND 4.0 (what root
`LICENSE` said at the time) or the differentiated per-package model a
concurrent restructuring had implemented in `packages/` — AGPL v3 for
Lumina, Starline Weaver, RDP, and Consent Transport; Proprietary for
CrystalBridge; a MIT-non-commercial/paid-commercial dual license for
CrystalCore-EI; CC BY-NC-ND for mythos. That restructuring went further
after ADR-0009 — a "Phase 4" commit added matching `license` fields to
every `packages/*/pyproject.toml`, making the differentiated model more
concretely real, not less.

Asked directly for a recommendation, the assessment was: the four-way split
is the kind of licensing strategy a company with legal counsel and paying
customers negotiates. For a pre-revenue, single-maintainer project, running
four different license regimes in parallel is real, ongoing operational
overhead — AGPL v3 in particular only provides value if violations can
actually be detected and pursued — for protection that isn't needed yet.
It also multiplies exactly the kind of drift that produced ADR-0009 in the
first place: more license regimes means more places for the next
uncoordinated session to get one of them wrong. The maintainer agreed:
**uniform CC BY-NC-ND, not the differentiated model.**

## Decision

- **The entire repository — `src/`, `packages/`, `mythos/` — is uniformly
  CC BY-NC-ND 4.0.** No per-package variation. Root `LICENSE` already said
  this for `src/` and mythos; this ADR extends it explicitly to `packages/`
  and closes the question ADR-0009 left open.
- **Every `packages/*/LICENSE.md` rewritten** from its previous license
  (AGPL v3 ×4, Proprietary ×1, MIT-non-commercial ×1) to CC BY-NC-ND 4.0,
  matching the root file's terms and copyright holder
  (Crystal Arena-Turner / TerAustralis Incognita, per
  [`ADR-0007`](ADR-0007.md)), each pointing back to this ADR for why its
  license changed twice in one day.
- **Every `packages/*/pyproject.toml`'s `license` field** updated to
  `CC-BY-NC-ND-4.0` to match, and the per-package `README.md`s' license
  badges/sections corrected (`crystalcore-ei/README.md` needed the most
  rework — it had a full MIT/Commercial installation and licensing
  section, not just a badge).
- **`LICENSE.md`, `LICENSING.md`, `COMMERCIAL_LICENSE.md`** (root files
  from the restructuring, actually specific to CrystalCore-EI despite
  reading as repo-wide) and **`docs/governance/LICENSING-STRATEGY.md`,
  `LICENSING-QUICK-REFERENCE.md`** — marked superseded in place, not
  deleted. They're real, detailed work describing a strategy that was
  seriously considered and concretely implemented before being reverted;
  preserving them (per
  [`Repository-Principles.md`](../governance/Repository-Principles.md) #3)
  keeps that reasoning available if the calculus changes later.
- **`README.md`'s license section rewritten again** — ADR-0009's version
  correctly described the situation as unresolved; this ADR resolves it,
  so the hedging ("in-progress," "not yet wired in") is replaced with a
  flat statement of the uniform license and a pointer to this ADR for the
  reverted alternative.

## Consequences

- One license, one file to check, for the whole repository — the
  simplicity this ADR explicitly traded for is now real, not aspirational.
- The differentiated model's specific protections are gone for now: no
  AGPL network-copyleft clause on the protocol-shaped packages (Starline
  Weaver, Consent Transport, RDP), no closed-source option for CrystalBridge,
  no sellable dual-license story for CrystalCore-EI. If a real reason
  appears later — an actual customer wanting to license CrystalCore-EI
  commercially, for instance — `LICENSING-STRATEGY.md`'s reasoning is
  still there to restart from, and that's a new ADR's job, not a reason to
  hold this one back.
- `packages/*/LICENSE.md` files have now changed license twice in the same
  day (AGPL v3/Proprietary/MIT → CC BY-NC-ND). Anyone who cloned or forked
  in between saw a different license than what's here now — an accepted
  cost of resolving the chaos quickly rather than leaving it open longer
  to be extra careful about a low-traffic pre-revenue repository.
- This is the fourth ADR in one day touching the code license
  (0006 → 0008 → 0009 → 0010). If a fifth uncoordinated session touches
  licensing again, this ADR — not any of the superseded strategy docs — is
  the one to check first.
