# Lumina – The Sovereign Companion

**Local-first AI companion with consent-driven memory and emotional intelligence.**

[![License: CC BY-NC-ND 4.0](https://img.shields.io/badge/license-CC%20BY--NC--ND%204.0-lightgrey.svg)](LICENSE.md)
[![Python 3.10+](https://img.shields.io/badge/python-3.10+-green.svg)](https://www.python.org/)

---

## Features

✨ **Local-first architecture** – All data stays on your machine  
🧠 **Emotional intelligence** – Understands and responds to emotions  
🔐 **Consent-driven** – You control what Lumina knows and remembers  
🎯 **Personalized** – Learns your preferences over time  
📱 **Multi-interface** – Terminal, web UI, API, voice  

---

## Installation

```bash
pip install teraaustralis-lumina
```

Or install in development mode:

```bash
git clone https://github.com/CrystalArchitect/teraaustralis-incognita.git
cd teraaustralis-incognita/packages/lumina
pip install -e .
```

---

## Quick Start

### Terminal Interface

```bash
lumina
```

### As a Python Library

```python
from teraaustralis.lumina import Lumina

companion = Lumina()
response = companion.chat("How are you feeling today?")
print(response)
```
